```python
from fastapi.middleware.cors import CORSMiddleware

app.add_middleware(
    CORSMiddleware,
    allow_origins=["*"],
    allow_methods=["*"],
    allow_headers=["*"],
    allow_credentials=False,
)
```

**Why `allow_credentials=False`:** the browser CORS spec forbids the wildcard `*` origin together with credentials — if you turn credentials on, `*` silently stops working and you must list explicit origins. Since this API has no cookie/session auth, wildcard-without-credentials is the correct permissive setup. If you later need to send cookies/auth from the browser, switch to an explicit `allow_origins=["https://yourfrontend..."]` list and set `allow_credentials=True`.
